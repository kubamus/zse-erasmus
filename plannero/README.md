# Plannero

### Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Database:** MySQL
- **ORM:** [Drizzle ORM](https://orm.drizzle.team/)
- **Authentication:** [Better Auth](https://better-auth.com/)
- **Styling:** Tailwind CSS
- **Validation:** Zod

To run this project, **[read the How to Run Tutorial](howToRun.md)**

## Scripts Overview

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the application for production.
- `npm run start` - Runs the built production application.
- `npm run db:push` - Pushes the Drizzle schema changes directly to the MySQL database.
- `npm run db:studio` - Opens the Drizzle Studio UI to browse the database.
